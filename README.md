# Unstuck Coach

Unstuck Coach is an executive-function accessibility coach for the moment when
someone has a real life pile and cannot turn intention into action.

It is built for task initiation, working-memory overload, calendar and inbox
reality, message shame spirals, re-entry after avoidance, body-state friction,
and closing loops without turning the user into a productivity problem.

## Try It First

- Live demo: <https://unstuck.kyanitelabs.tech/>
- Landing page: <https://simongonzalezdc.github.io/unstuck-coach-week5/landing/>

First prompt:

```text
I need a coach to get started on this.
```

The coach should not ask you to organize the mess first. It should accept the
messy pile, hold the extra context, and give back one small next move.

## What Makes It Different

Unstuck is not a productivity article generator. A good reply has three jobs:

1. Name what is happening in plain language, without blame.
2. Choose one concrete next move that fits the user's current state.
3. Ask for one tiny proof answer the user can give without thinking hard.

It treats food, rest, messages, bills, inboxes, calendars, transitions, and
restart breadcrumbs as real executive-function access needs.

## Safety

This is a coaching scaffold, not medical care or therapy. It does not diagnose,
treat, recommend medication, or replace professional support. Crisis and
clinical boundaries are in [reference/safety-boundaries.md](reference/safety-boundaries.md).
